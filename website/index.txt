h1. Code smell detector

h1. &#x2192; 'reek'


h2. What

Reek is a tool that examines Ruby classes, modules and methods and reports any code smells it finds.

h2. Installing

<pre syntax="ruby">sudo gem install reek</pre>

h2. The basics

<pre syntax="ruby">
  $ cd my_project/lib
  $ reek
</pre>

(See reek --help for more.)

h2. Code Smells

reek currently includes very naive checks for the following code smells:

* Long Method
* Large Class
* Feature Envy
* Uncommunicative Name
* Long Parameter List
* Utility Function
* Nested Iterators
